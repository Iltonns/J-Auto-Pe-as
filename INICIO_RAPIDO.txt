┌──────────────────────────────────────────────────────────────────────┐
│                   🚀 COMECE AGORA (10 MINUTOS)                      │
└──────────────────────────────────────────────────────────────────────┘

  PASSO 1: Configure o Neon (5 min)
  ┌────────────────────────────────────────────────────────────────┐
  │ 1. Acesse https://neon.tech                                    │
  │ 2. Crie um novo projeto                                        │
  │ 3. Copie a connection string                                   │
  │    (postgresql://usuario:senha@host/database?sslmode=require)  │
  └────────────────────────────────────────────────────────────────┘

  PASSO 2: Configure o .env (2 min)
  ┌────────────────────────────────────────────────────────────────┐
  │ Abra o arquivo .env e cole:                                    │
  │                                                                 │
  │ DATABASE_URL=postgresql://[COLE_AQUI]                          │
  │ SECRET_KEY=troque-por-algo-super-aleatorio                     │
  │ FLASK_ENV=production                                           │
  │                                                                 │
  │ 💡 Use o arquivo CONFIGURAR_NEON.txt como guia!               │
  └────────────────────────────────────────────────────────────────┘

  PASSO 3: Instale Dependências (1 min)
  ┌────────────────────────────────────────────────────────────────┐
  │ Execute no PowerShell:                                         │
  │                                                                 │
  │ pip install -r requirements.txt                                │
  └────────────────────────────────────────────────────────────────┘

  PASSO 4: Inicie o Sistema (30 seg)
  ┌────────────────────────────────────────────────────────────────┐
  │ Execute no PowerShell:                                         │
  │                                                                 │
  │ python app.py                                                  │
  │                                                                 │
  │ O sistema criará todas as tabelas automaticamente!            │
  └────────────────────────────────────────────────────────────────┘

  PASSO 5: Teste (2 min)
  ┌────────────────────────────────────────────────────────────────┐
  │ 1. Acesse: http://localhost:5000                              │
  │ 2. Login: admin / admin123                                     │
  │ 3. Teste: cadastro, venda, relatórios                         │
  └────────────────────────────────────────────────────────────────┘
